# Meds/ManualLysis

This project is a unified application serving two primary purposes:
1. **Manual Lysis Protocol**: A clinical reference tool built originally as a standalone component.
2. **MedDoses**: A centralized cheat sheet and utility view for MedDoses features.

## Architecture
The application uses a state-driven layout with a sidebar navigation component to easily switch between the Manual Lysis viewer and the Developer Hub.

## Getting Started
To run the local development server:
```bash
npm install
npm run dev
```
